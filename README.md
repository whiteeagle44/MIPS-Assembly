# Short programs in C & MIPS Assembly 

## Task 1
Change digits in a string for their complements.

### Example

* 9 -> 0  (9-9=0)
* 8 -> 1  (9-8=1)
* 4 -> 5  (9-4=5)

### Comment  

Each digit should be changed to  '9' - digit + '0'

In file `1.c`
